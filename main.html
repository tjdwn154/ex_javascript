Web

날짜 : 2021년 11월 31일 (출처 : 한빛아카데미)



웹은 인터넷 공간에서 제공하는 서비스이다.
영국의 컴퓨터 과학자 팀 버너스리가 1989년 인터넷에서 문서 사이를 쉽게 이동할 수 있는 하이퍼 링크를 제안하였고, 1991년 웹을 발표하였다.
팀 버너스리는 1993년 웹 소스 코드를 무료로 공개하였으며, 이후 자신이 고안해낸 여러 기술들을 특허를 내지 않고 무료로 공개하고 1994년 W3C(World Wide Web Consortium)을 창설하여 인터넷이 더 빠르게 발전 하는데 큰 기여를 하였다.

HTML5

날짜 : 2021년 11월 31일 (출처 : 한빛아카데미)



웹 표준 기술을 총칭하며 CSS3와 JavaScript를 모두 포함한다.
작은 의미로는 웹 페이지를 구성하는 HTML 마크업 언어 자체이다. 애플리케이션 수준의 웹 페이지를 만들 수 있고, 데스크탑 애플리케이션을 만들 수 있으며 리액트 네이티브를 사용해 모바일 애플리케이션을 만들 수 있다.

CSS3

날짜 : 2021년 11월 31일 (출처 : 위키백과)



마크업 언어가 실제 표시되는 방법을 기술하는 스타일 언어(style sheet languange)로 HTML과 XHTML에 주로 쓰이며, W3C의 표준이다. 마크업 언어가 웹 사이트의 몸체를 담당한다면 CSS는 옷과 액세사리처럼 꾸미는 역할을 담당한다고 할 수 있다.
CSS3의 경우 그림자 효과, 그라데이션, 변형 등 그래픽 편집 프로그램으로 제작한 이미지를 대체할 수 있는 기능이 추가되었다.
